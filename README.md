# 💰 SmartSpend – Expense Tracker

A simple and interactive **Expense Tracker Web App** built using **HTML, CSS, and JavaScript**.
This application helps users track their income and expenses while displaying their current balance in real time.

## 🚀 Live Demo

🔗 https://manasign.github.io/smartspend/

## 📌 Features

✔ Add income and expense transactions <br>
✔ Automatically calculates total **balance** <bR>
✔ Displays **total income and expenses** separately <br>
✔ Clean and simple user interface <br>
✔ Instant updates when a new transaction is added

## 🛠️ Built With

* **HTML** – Structure of the web application
* **CSS** – Styling and layout
* **JavaScript** – Logic and dynamic functionality

## ⚙️ How It Works

1. Enter a **description** for the transaction.
2. Enter the **amount**.
3. Use **positive values for income** and **negative values for expenses**.
4. Click **Add Transaction**.
5. The app automatically updates:

   * Total Balance
   * Income
   * Expenses

## 💡 Example

| Description | Amount |
| ----------- | ------ |
| Salary      | +5000  |
| Groceries   | -800   |
| Movie       | -300   |

Balance will update automatically.
